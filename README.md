# nvim_config
Neovim configuration (uses NVChad as base)

# Remove old nvim config
```
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
```

# Base setup nvchad
see https://nvchad.com/docs/quickstart/install

```
git clone https://github.com/NvChad/NvChad ~/.config/nvim --depth 1
```
# Clone this repository as custom config
```
git clone https://github.com/jammo-nl/nvim_config ~/.config/nvim/lua/custom --depth 1
```
